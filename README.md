# Acadlog - College Management System

Acadlog is the all-in-one digital operating system for Indian colleges. Automate attendance, streamline grading, and deliver world-class hybrid education.

## Features

- **Automated Attendance**: Track student attendance seamlessly
- **Streamlined Grading**: Efficient grading and assessment system
- **Hybrid Education**: Support for both online and offline learning
- **Multi-language Support**: Available in English and Hindi
- **Modern UI**: Built with React and Tailwind CSS for a smooth user experience

## Tech Stack

- **Frontend**: React 19, TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Animations**: GSAP
- **Icons**: Heroicons
- **Fonts**: Plus Jakarta Sans, Space Grotesk
- **Utilities**: React Intersection Observer

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`
